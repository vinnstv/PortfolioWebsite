Face Detection & Recognition using LBPH and OpenCV

This project demonstrates a face recognition system I built using the Local Binary Pattern Histogram (LBPH) algorithm with OpenCV. LBPH is a simple and efficient algorithm for face recognition that operates by summarizing facial features into a histogram, making it robust in various lighting conditions. The project utilizes a dataset of celebrity faces, and I trained a model capable of identifying individuals from new, unseen images.
